Ключевые решения по дизайну

Hero — GitHub-style Globe

Three.js + ~12k InstancedMesh кружков для суши. SVG-заглушка → crossfade к WebGL. Реальные TLE-данные спутников «Рассвет» для орбитальных дуг. Глобус занимает правую половину экрана (как у Cloudflare).

Scroll animations — GSAP ScrollTrigger

Apple-style: текст появляется снизу, секции «прилипают» (scrub), счётчики цифр анимируются при входе в viewport. Pinned sections для технологий и хронологии.

Технологии — горизонтальный scroll pin

Секция «прикрепляется» к экрану, карточки технологий проезжают горизонтально при скролле (как на apple.com/iphone).

Хронология — вертикальный scroll-driven

Линия таймлайна «рисуется» по мере скролла. Каждое событие появляется при достижении его позиции.

Lighthouse 90+ стратегия

Three.js загружается dynamic import только когда глобус входит в viewport. GSAP — тоже lazy. Astro отправляет 0KB JS по умолчанию. SVG-заглушка обеспечивает мгновенный LCP.
